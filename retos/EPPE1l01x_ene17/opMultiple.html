A. Se deberá instalar una planta de generación con base a combustión de biomasa agrícola residual que se deberá de traer desde cientos de kilómetros de distancia. La combustión generará vapor de agua para mover una turbina de vapor y un generador de energía eléctrica.
B. Se deberá instalar una planta de generación termosolar basada en concentradores solares. Los concentradores solares calientan agua, la cual es escaza y tendrá que transportarse de lejos, para generar vapor de agua y mover una turbina de vapor y un generador de energía eléctrica.
C. Se deberá instalar una planta fotovoltáica para convertir la energía solar en energía eléctrica, la cual se puede inyectar a la red eléctrica. Asimismo, se contará con un banco de almacenamiento de energía eficiente para suministrar la energía a la red cuando sea conveniente.
D. Se deberá instalar una granja eólica para convertir la energía del viento en energía eléctrica, la cual se puede inyectar a la red eléctrica. Como el viento es muy variable, se tendría que tener una planta de generación de respaldo que queme combustóleo el cual es contaminante y se tendría que transportar desde lejos.
